# 🔧 环境配置

## 系统要求

| 环境 | 版本要求 |
|------|----------|
| **Python** | 3.10+ |
| **Node.js** | 18+ |
| **pnpm** | 8+ |

## 后端环境 (Python)

确保已安装 Python 3.10 或更高版本：

```bash
python --version  # 应显示 3.10+
```

推荐使用 **uv** 作为包管理器（更快）：

```bash
# 安装 uv
pip install uv
```

## 前端环境 (Node.js)

确保已安装 Node.js 18 或更高版本：

```bash
node --version  # 应显示 18+
```

安装 pnpm 包管理器：

```bash
npm install -g pnpm
```

## 数据库

开发环境必须使用 **MySQL** 或 **PostgreSQL** 数据库。

::: warning 注意
不支持 SQLite，请确保已安装并配置好 MySQL 或 PostgreSQL。
:::
