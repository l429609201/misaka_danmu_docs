# ✂️ 弹幕编辑

弹幕编辑功能允许您对已导入的弹幕进行精细化调整，包括拆分、合并和时间偏移操作。

> 此功能自 v2.5.0 起支持，需要弹幕源 ≥ 2.0.0 版本

![弹幕操作-弹幕编辑](/screenshot/弹幕操作-弹幕编辑.png)

---

## 功能概览

| 功能 | 说明 |
|------|------|
| **拆分弹幕** | 将一集的弹幕按时间点拆分为多个部分 |
| **合并弹幕** | 将多集的弹幕合并为一集 |
| **偏移弹幕** | 调整弹幕的时间轴，解决弹幕与视频不同步的问题 |

---

## 使用方法

### 进入弹幕编辑

1. 在 **弹幕库** 中选择目标条目
2. 进入分集详情页
3. 点击 **"弹幕编辑"** 按钮

---

## 拆分弹幕

当一个弹幕文件包含了多集内容时（如合集视频），可以使用拆分功能将其分开。

### 操作步骤

1. 选择需要拆分的分集
2. 点击 **"拆分"** 操作
3. 设置拆分时间点（支持设置多个拆分点）
4. 预览拆分结果
5. 确认拆分

### 适用场景

- 合集视频的弹幕需要按集拆分
- 弹幕文件包含片头/片尾需要去除
- 长视频弹幕需要按章节分割

---

## 合并弹幕

将多个分集的弹幕合并为一个，适用于需要将分散的弹幕整合的场景。

### 操作步骤

1. 选择需要合并的多个分集
2. 点击 **"合并"** 操作
3. 设置合并顺序
4. 预览合并结果
5. 确认合并

### 适用场景

- 将上下集弹幕合并为完整版
- 整合来自不同来源的同一集弹幕
- 合并分段视频的弹幕

---

## 偏移弹幕

调整弹幕的时间轴偏移量，解决弹幕与视频画面不同步的问题。

### 操作步骤

1. 选择需要调整的分集
2. 点击 **"偏移"** 操作
3. 输入偏移时间（正值为延后，负值为提前）
4. 预览偏移效果
5. 确认偏移

### 适用场景

- 弹幕与视频存在固定时间差
- 不同版本视频的片头时长不同
- 弹幕源与本地视频的时间轴不一致

---

## 注意事项

::: warning 重要提示
- 弹幕编辑操作会直接修改已存储的弹幕数据，建议操作前先备份
- 此功能需要弹幕源版本 ≥ 2.0.0，且弹幕源 2.0.0 版本开始不向下兼容
- 编辑后的弹幕会立即生效，客户端下次请求时会获取到更新后的弹幕
:::
