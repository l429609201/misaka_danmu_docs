# 🌏 bili 源地区限制解除

由于版权原因,B站的部分番剧和影视内容存在地区限制,港澳台地区的部分内容在中国大陆无法直接访问。本文档将指导您如何通过配置解析服务器来突破这些限制。

![bilibili源配置代理访问限制地区内容](/screenshot/bilibili源配置代理访问限制地区内容.png)

::: tip 适用范围
本功能仅支持 **港澳台地区** 的番剧和影视内容解析,因为泰区、东南亚地区没有开放弹幕功能。
:::

## 🎯 功能说明

### 什么是地区限制?

- **大陆限制内容**: 部分番剧和影视作品由于版权原因,仅在港澳台地区可观看
- **搜索问题**: 直接搜索这些内容可能无法找到或无法获取弹幕
- **播放限制**: 即使找到内容,也可能因地区限制无法播放

### 解析服务器的作用

通过配置解析服务器,系统会:
1. **代理请求**: 将 B站 API 请求通过解析服务器转发
2. **突破限制**: 绕过地区检测,访问港澳台地区专属内容
3. **获取弹幕**: 正常获取和显示弹幕信息
4. **搜索番剧**: 搜索并匹配港澳台地区的番剧资源

## 📋 前置准备

在开始配置之前,您需要:

- ✅ 一个可用的解析服务器地址(自建或使用公共服务器)
- ✅ 确保网络能够访问解析服务器
- ✅ bili 弹幕源已正确安装并启用

## 🔧 配置解析方法

1. **进入配置页面**
   - 打开 Web UI
   - 导航至 **弹幕源管理** → **bili 源配置**

   -（1）port1-配置解析服务器

      ```
      解析服务器地址: https://your-server.example.com
      ```

   -（2）port2-配置本地代理配置全局HK
   
      1.勾选限制内容本地代理选项

      >PS:勾选之后解析服务器将无法使用

      2.添加本地代理
      
      ```
      本地代理: http://127.0.0.1:7890
      ```

3. **保存并测试**
   - 点击 **确认** 按钮保存

4. **搜索测试**
   - 在搜索框中输入港澳台地区番剧名称
   - 如 "死神少爷与黑女仆"
   - 查看是否能正常搜索到结果并获取弹幕

![bili源-区域限制内容死神少爷与黑女仆搜索效果图](/screenshot/bili源-区域限制内容死神少爷与黑女仆搜索效果图.png)

## 🏗️ 解析服务器获取

### 一、自建解析服务器(推荐)

参考 [BiliRoaming](https://github.com/yujincheng08/BiliRoaming) 项目WIKI[自建解析服务器](https://github.com/yujincheng08/BiliRoaming/wiki/%E8%87%AA%E5%BB%BA%E8%A7%A3%E6%9E%90%E6%9C%8D%E5%8A%A1%E5%99%A8)提供的方式和接入方法搭建自建服务:

>虽然自建服务提供了两个参考的搭建，但是二者均已归档，现在使用另一个自建解析项目搭建，使用[vercel](https://vercel.com/)可轻松部署使用。

1. **fork大佬制作好的解析项目**

 [biliroaming-ts-server-vercel](https://github.com/bili-vd-bak/biliroaming-ts-server-vercel)

2. **部署项目**
 - （1）登录到[vercel](https://vercel.com/) 链接第1步[github](https://github.com/)fork的仓库
 - （2）部署项目到vercel运行起来
 - （3）将得到的访问地址复制到浏览器中，如出现以下截图则为部署成功
![自建代理服务器部署成功后浏览器访问截图.png](/screenshot/自建代理服务器部署成功后浏览器访问截图.png)
 >如果vercel提供的域名无法访问，可能是被墙了，可通过把访问域名加入代理或者更换域名
 >（6位数字域名一年很便宜）解决

3. **配置弹幕源**
  将部署成功后可访问的域名填写到bili源代理服务器中，即可进行地区限制内容搜索并获取弹幕


### 二、使用公共解析服务器

::: warning 注意
- 公共服务器可能不稳定或突然失效
- 请勿将公共服务器地址用于商业用途或大规模传播
- 建议优先使用自建服务器以确保服务稳定性
:::


> 更多公共服务器信息可参考 [BiliRoaming Wiki](https://github.com/yujincheng08/BiliRoaming/wiki/%E5%85%AC%E5%85%B1%E8%A7%A3%E6%9E%90%E6%9C%8D%E5%8A%A1%E5%99%A8),但请注意 **禁止宣传** 的使用规范。

## 🔍 使用说明

### 搜索港澳台番剧

1. **直接搜索**
   - 在搜索框中输入番剧中文名或日文名
   - 系统会自动通过解析服务器搜索

2. **获取到正确的搜索结果**
   - 港澳台地区的番剧通常会标注地区信息
   - 注意区分与大陆版本的差异(集数、字幕等)

## 🛠️ 故障排除

### 无法连接解析服务器

**症状**: 提示 "解析服务器连接失败"

**解决方案**:
1. 检查服务器地址是否正确
2. 使用 `curl` 或浏览器测试服务器可用性:
   ```bash
   curl -I https://your-server.example.com
   ```
3. 确认防火墙没有阻止连接
4. 尝试更换其他解析服务器

### 搜索不到港澳台番剧

**症状**: 搜索结果为空或只显示大陆版本

**解决方案**:
1. 确认解析服务器已正确配置并启用
2. 检查日志查看是否有错误信息
3. 尝试使用完整的番剧名称搜索
4. 确认该番剧在港澳台地区确实有版权

## 🔐 隐私与安全

### 数据传输

- 所有请求都通过解析服务器中转
- **自建服务器**: 数据完全由您控制,安全性最高
- **公共服务器**: 数据会经过第三方服务器,存在一定隐私风险

### 建议

1. ✅ **优先使用自建服务器** 保护隐私
2. ✅ **配置 HTTPS** 确保数据传输加密
3. ✅ **定期更换公共服务器** 降低风险
4. ✅ **不要在公共服务器上传输敏感信息**

### 缓存配置

启用缓存可以减少对解析服务器的请求:

```json
{
  "cache": {
    "enabled": true,
    "ttl": 3600,
    "maxSize": 1000
  }
}
```
